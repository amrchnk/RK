Проект "Личный кабинет студента и преподавателя Московского Политеха"


Цель проекта

Проведение редизайна сайта Личного кабинета студента и преподавателя Московского Политеха. По итогу работы должен быть получен готовый прототип с дизайном и переходами по сайту.

Заказчик проекта

Федеральное государственное бюджетное образовательное учреждение высшего образования «Московский политехнический университет»

Этапы работы над проектом

Весенний семестр 2019-2020
  1. Разработка нового дизайна для десктопной версии сайта для преподавательской и студенческой частей
  2. Верстка страниц для десктопной версии сайта для преподавательской и студенческой частей
  3. Разработка скриптов и сборка проекта
Осенний семестр 2020-2021
  1. Внесение изменений в дизайн для десктопной версии сайта
  2. Верстка страниц для десктопной версии сайта согласно внесенным изменениям
  3. Адаптация нового дизайна для мобильной версии сайта
  4. Верстка страниц для мобильной версии сайта
  5. Перенос проекта на TypeScript
  6. Разработка скриптов и сборка проекта
Весенний семестр 2020-2021
  1. Тестирование прототипа сайта, выявление ошибок
  2. Отладка front-end'a сайта
  3. Отладка back-end'a сайта
  4. Выход проекта на голосование вместе с другими прототипами ЛК на официальной странице Московского Политеха в ВК и Телеграме
  5. Тестирование прототипа будущими пользователями, получение отзывов
  6. Кооперация с другими командами, анализ отзывов, определение нового вектора развития проекта
